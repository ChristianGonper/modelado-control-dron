# Reference Scenarios v1

Versioned external scenario artifacts used for phase 4 validation.

Files:
- `hover.json`: nominal hover regression.
- `angular_maneuver.json`: circular tracking regression with attitude changes.
- `perturbation.json`: hover with wind gusts and observation noise.
- `dt_comparison_coarse.json`: coarse physics discretization.
- `dt_comparison_fine.json`: fine physics discretization.
